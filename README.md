# Badge Forge — SVG Custom Badge Generator

Полностью клиентский генератор SVG-бейджей для README и документации.

## Что это

`Badge Forge` теперь работает только в браузере:
- без Flask/API на сервере;
- без эндпоинтов `/badge`, `/catalog`, `/health`;
- подходит для статического хостинга (включая Vercel).

## Локальный запуск

Откройте `index.html` напрямую в браузере
или поднимите любой статический сервер:

```bash
python -m http.server 8080
```

После этого откройте `http://localhost:8080`.

## Деплой на Vercel

Проект деплоится как статический сайт.

```bash
vercel
vercel --prod
```

`vercel.json` настроен на отдачу `index.html` на корне сайта.

## Структура

```text
.
├── index.html        # UI Badge Forge
├── app.js            # Клиентский движок генерации SVG
├── styles.css        # Стили интерфейса
├── vercel.json       # Конфиг статического деплоя для Vercel
└── sample_*.svg/     # Примеры бейджей
```

## Примечание

Серверный файл `api/stats_index.py` удален: генерация бейджей выполняется полностью на клиенте.
